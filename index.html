Smartfarm Monotoring

Smartfarm Monitoring adalah sebuah website pemantauan kondisi lingkungan pertanian dengan IoT. Petani atau user dapat memantau kondisi lingkungan seperti kelembaban udara, intensitas cahaya, suhu dan lainnya secara realtime yang di tampilkan dengan line chart. Website ini pada bagian admin juga memiliki mekanisme CRUD menggunakan Laravel arsitektur MVC, seperti menambah sensor, menghapus data lahan, dan masih banyak lagi. Saya sebagai Back-End Developer. Melakukan Fetch data untuk proses CRUD dalam framework Laravel dan pembuatan RESTful API untuk website tersebut menggunakan Native PHP.
